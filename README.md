Color Match Game

A simple web-based color match game built with **HTML, CSS, and JavaScript**.

How to Play

1. Click **Start Game**.
2. A color name will appear in a random text color.
3. Click the button that matches the **color of the text**, not the word itself.
4. Your **score increases** for every correct click.
5. Try to get the highest score possible!

Files

- `index.html` → Main HTML file  
- `style.css` → Styles for layout and buttons  
- `script.js` → JavaScript logic for the game  

Features

- Random color name and text color each round  
- Score tracking for correct answers  
- Interactive and easy to play  

Deployment

You can deploy this game using **Vercel**:

1. Go to [Vercel](https://vercel.com/) → New Project → Import Git Repository  
2. Select this repository  
3. Framework Preset: **Other (Static Site)**  
4. Root Directory: leave empty  
5. Output Directory: leave empty  
6. Click **Deploy**  

After deployment, you’ll get a live URL to play the game online.
